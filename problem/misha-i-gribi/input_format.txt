На вход поступает 3 целых числа разделённые пробелами.

Первое число - точная оценка Миши вида гриба (1 - съедобный, 0 - несъедобный, -1 - ядовитый).

Второе число - волюнтаристская оценка Маша вида гриба (1 - съедобный, 0 - несъедобный, -1 - ядовитый).

Третье число - характеризует любит ли миша данный вид грибов (1 - любит, -1 - не любит).
